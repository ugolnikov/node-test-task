# Тестовое задание Node.js

REST API для управления пользователями

## Технологии

- Node.js + Express
- TypeScript
- Prisma ORM (SQLite)
- JWT Auth
- Postman

## Установка

```bash
npm install
```

## Настройка

.env:

```bash
DATABASE_URL="file:./dev.db"
JWT_SECRET=your-secret-key
PORT=3000
```

## Запуск

- **Разработка**: `npm run dev`
- **Проект**: `npm start`
